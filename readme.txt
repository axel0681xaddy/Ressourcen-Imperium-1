# Ressourcen-Imperium: IDLE Game

## Projektstruktur
- index.html: Hauptseite
- css/styles.css: Styling
- js/: JavaScript-Module
  - main.js: Haupteinstiegspunkt
  - gameState.js: Spielzustand
  - gameLoop.js: Hauptspielschleife
  - display.js: UI-Updates
  - events.js: Event-Handler
  - storage.js: Speichern/Laden
  - upgrades.js: Freischaltlogik
  - achievements.js: Errungenschaften
  - prestige.js: Prestige-System

## Funktionen
- Ressourcensammlung: Klicken und automatische Produktion
- Mehrere Ressourcentypen mit Konvertierungen
- Gebäude und Upgrades
- Prestige-System
- Errungenschaften


Beschreibung für neuen Chat: 

Wir haben ein IDLE-Game namens 'Ressourcen-Imperium' entwickelt mit mehreren Ressourcentypen, Produktionsketten und einem Prestige-System. Der Code ist modular aufgebaut mit separaten JavaScript-Dateien für verschiedene Funktionen.
